Publish는 변경 내용을 발행하는 것입니다.
현재의 변경내용은 아직 로컬 저장소의 head 안에 있습니다.
이제 이 변경 내용을 원격 서버로 올려봅시다. 

publish의 명령어에는 이런 것들이 있습니다. 

 * [git push]()
 * [git push remote]()
 * [git format-patch origin]()



1. git push
```
$ git push
```

2. git push remote
```
$ git push remote
```
3. git format-patch origin
```
$ git format-patch origin
```
